## Before New Game
- Set Options to Fast Text, Battle Animations Off, Battle Style Set

## Intro
- Name yourself and your rival a one character name
- Walk into the grass to trigger the Pikachu cutscene
- Don't nickname Pikachu

  - Geodude: DK x2
